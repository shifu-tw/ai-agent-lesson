# 每週沉澱

> 等你穩定使用兩週後再開啟。

## 排程

每週日 21:00

## 任務

### 1. 掃描記憶
- 讀 MEMORY.md 所有條目
- 哪些被反覆提到？（高頻 = 重要）
- 有沒有跟 CLAUDE.md 矛盾的？
- 有沒有過時的記憶該清理？

### 2. 掃描知識庫
- vault/ 本週新增了什麼？
- 產出簡短摘要

### 3. 找模式
- 最近的對話有什麼重複出現的需求？
- 如果某類要求出現 ≥ 3 次 → 建議建 Skill
- 偏好有沒有改變？ → 建議更新 CLAUDE.md

### 4. 產出沉澱日誌

存到 vault/reviews/YYYY-MM-DD-weekly.md：

```markdown
# 週沉澱 YYYY-MM-DD

## 本週記憶摘要
- ...

## 高頻模式
- ...

## 建議
- [ ] 更新 CLAUDE.md 的 ___（原因：___）
- [ ] 建新 Skill：___（原因：出現 N 次）
- [ ] 清理記憶：___（原因：過時）

## 等待你的確認
以上建議需要你批准才會執行。
```
