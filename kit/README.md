# AI 夥伴安裝包

這是你的 AI 夥伴的完整框架。所有設計都圍繞一個原則：**讓 AI 有限的工作記憶裡，永遠放著最重要的東西。**

## 架構

```
核心（今天建好）
  CLAUDE.md   — 身份設定。每次對話自動載入。
  MEMORY.md   — 記憶索引。自動累積，你可以看、改、刪。
  進化規則     — 寫在 CLAUDE.md 裡。讓它自己變好。

擴展（用到時加）
  vault/      — 知識庫。按需搜尋，不預載。
  skills/     — 專長。用到才載入。
  CLI 串接    — 程式取資料，只把結果餵 AI。

自動化（穩定後開）
  routines/   — 排程任務。每日晨報、每週沉澱。
```

## 快速開始

1. 跟 Agent 說：**「讀 guides/01-setup.md 然後幫我跑安裝」**
2. 安裝完後說：**「讀 guides/02-onboarding.md 然後幫我跑 onboarding」**
3. 跑完就有一個認識你的 AI 夥伴了

## 資料夾說明

| 資料夾 | 用途 | Context 策略 |
|--------|------|-------------|
| `vault/` | 知識庫 | 按需搜尋，不預載 |
| `skills/` | 可重複用的 SOP | 觸發時才載入 |
| `routines/` | 排程任務 | 獨立執行 |
| `guides/` | 建造引導 | 用完可不管 |
| `tools/` | 程式工具 | 程式跑，零 token |
| `milestones/` | 進度追蹤 | 驗收時用 |
| `references/` | 深入知識 | Agent 搜尋用 |

## 進化邏輯

```
日常對話 → 記憶累積 → 發現重複 → 建議建 Skill
                    → 每週沉澱 → 建議更新身份
                    → 你批准 → 系統升級 → 繼續對話
```
