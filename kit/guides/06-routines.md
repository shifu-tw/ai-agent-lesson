# 排程任務引導

## 什麼是 Routine？

Routine 是讓 AI 不用你開口就自己做事。
你設好時間和任務，它到時間就自動跑。

**不建議第一天就開。** 先讓它認識你至少一兩週，累積足夠的記憶和脈絡，再開自動化才有品質。沒有脈絡的自動化只是噪音。

## 預建的 Routine 範例

### 每日晨報

```
每天早上 9:00
1. gws gmail +triage → 讀信
2. gws calendar +agenda → 看行事曆
3. 掃 vault/ 最近新增的檔案
4. 結合 CLAUDE.md 的脈絡，產出：
   - 今天最該注意的 3 件事
   - 需要準備的會議
   - 你可能會想知道的新資訊
```

### 每週沉澱

```
每週日晚上 9:00
1. 掃描 MEMORY.md → 這週記住了什麼？
   - 哪些被反覆提到？（高頻 = 重要）
   - 有沒有跟 CLAUDE.md 矛盾的？
2. 掃描 vault/ → 新增了什麼？整理摘要
3. 回顧對話模式 → 有沒有重複的需求？
   - 如果有 → 建議建 Skill
4. 產出沉澱日誌 → vault/reviews/YYYY-MM-DD.md
   - 本週學到的
   - 建議更新的內容
   - 等你確認後才修改 CLAUDE.md
```
