# 第一週：用爆它

## 唯一規則

**本來就在做的事，全部先問它。**

- 收到信 → 先讓它讀
- 開完會 → 跟它說剛聊了什麼
- 要寫東西 → 讓它先起草
- 做決策前 → 把脈絡丟給它，問它看到什麼你沒看到的
- 讀到文章 → 丟進 vault/
- 想到什麼 → 直接跟它說

## Context 管理提醒

- 不同任務開不同對話
- 長對話品質會衰退，重要任務開新的
- 兩段式：先討論產出文件，再開新對話照文件執行

## 一週後驗收

跟 Agent 說「跑 review」。

### 基本門檻

- [ ] MEMORY.md ≥ 15 條記憶
- [ ] vault/ ≥ 5 份文件
- [ ] 至少起草 3 份內容
- [ ] 至少整理 3 場會議
- [ ] gws 至少用 10 次
- [ ] 問「你了解我什麼？」準確 ≥ 10 點
- [ ] 它寫的東西 60% 以上不用大改
- [ ] 至少糾正 3 次且沒再犯
- [ ] 開始習慣「先問它一下」
- [ ] 能說出「它最懂我的一件事」是什麼
